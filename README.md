## 🎮 游戏介绍

这是基于《大明王朝1566》知识库打造的沉浸式权谋RPG。你可以选择8个历史角色，在改稻为桑、毁堤淹田、严党倒台等历史节点做出抉择，体验大明嘉靖年间的权力博弈。

**核心特色：**
- ✨ 8个可选角色，每个有独特属性和绝技
- 🎯 7个章节的多线剧情
- ⚔️ 实时战斗系统（台州大战）
- 🏛️ 势力关系系统（帝心、严党、清流、内廷、军方、民心）
- 💎 8种不同结局，你的选择决定历史走向
- 📚 298个知识单元支撑，历史细节满满

## 🎯 玩法说明

### 游戏系统

**属性系统：**
- 📊 **声望 (Rep)** - 决定朝堂影响力
- 💰 **银两 (Gold)** - 用于贿赂、装备、军粮
- ❤️ **生命 (HP)** - 归零即游戏结束

**势力关系：**
- 🏛️ **帝心** - 皇帝的好感度
- 🦊 **严党** - 严嵩派系的好感度
- 🌊 **清流** - 徐阶等清派的好感度
- 🏮 **内廷** - 司礼监的好感度
- ⚔️ **军方** - 戚家军等的好感度
- 👥 **民心** - 百姓的认可度

**判定系统：**
- 政治判定：影响官场策略成功率
- 武力判定：影响战斗和威慑
- 智谋判定：影响计谋和布局
- 清廉判定：影响道德抉择
- 魅力判定：影响交际和说服

### 战斗系统

台州大战中采用回合制战斗：
- 猛攻：正面硬刚，高伤害
- 奇袭：包抄合围，可能全歼
- 救民：先救人质，民心大增
- 火攻：焚烧战船，风险高回报高

## 📚 知识库

本游戏基于《大明王朝1566》知识库构建，包含：

```
batch-0220-ming/
├── skus/                  # 知识单元（SKU）
│   ├── factual/          # 175个事实型知识
│   │   ├── 财政数据       # 嘉靖三十九年收支、改稻为桑数据
│   │   ├── 人物关系       # 主要官员、太监、商人
│   │   ├── 历史事件       # 毁堤淹田、台州大战、海瑞上疏
│   │   └── 制度器物       # 廷杖、锦衣卫、内阁票拟
│   ├── procedural/       # 123个程序型知识
│   │   ├── 政治策略       # 权谋博弈、官场自保、分化拉拢
│   │   ├── 军事战术       # 鸳鸯阵、伏击、侦察
│   │   ├── 刑罚制度       # 廷杖、诏狱、三法司审理
│   │   └── 宫廷礼仪       # 召见、圣旨、洗脚仪式
│   └── relational/       # 关系型知识
│       └── 标签树+术语表
├── eureka.md             # 跨领域洞察（权谋模式）
├── mapping.md            # SKU路由映射
└── game.html             # 游戏程序
```

**权谋模式（Eureka洞察）：**
- 合法性距离的设计
- 仪式即信息路由协议
- 符号意义的战略重构
- 边缘节点的代理控制
- 符号劫持的合法性透支
- 测试-授权-测试的信任验证循环
- 豁免权的双向编码
- 刚性代理的冲突升维机制
