# 明朝内阁票拟批红流程

## 概述
明朝中央政府处理政务文书的标准程序，涉及内阁拟票、司礼监批红、皇帝裁决等环节。

## 步骤

1. **各部提出方案**
   - 各部衙门根据实际需要拟定用款或办事方案
   - 详细列出数额、用途、理由
   - 呈报内阁

2. **内阁商议拟票**
   - 首辅召集阁员会议
   - 各部堂官说明情况
   - 内阁商议后拟定票拟（处理意见）
   - 首辅在票拟上署名

3. **奏请皇帝**
   - 首辅携带票拟入宫面圣
   - 向皇帝逐条奏陈各项事宜
   - 回答皇帝询问

4. **皇帝裁决**
   - 皇帝听取奏陈后做出决定
   - 可当场准奏或驳回
   - 可要求重新拟定

5. **司礼监批红**
   - 皇帝准奏后，由司礼监掌印太监用朱笔批红
   - 批红后方为正式生效的诏令

6. **执行落实**
   - 各部按批红后的诏令执行
   - 户部拨款、工部施工等

## 决策点
- 若皇帝不同意某项票拟，内阁需重新商议调整
- 若涉及重大事项，可能需要多次奏陈
- 各部之间若有分歧，需在内阁会议中协调

## 预期结果
政务文书经过完整程序后生效，各部据此执行具体事务。
