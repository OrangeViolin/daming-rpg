# 灾荒借粮赈济程序

## 概述
当官府赈灾粮不足时，知县可以县衙名义向本县大户借粮，日后归还。

## 步骤

1. **评估需求**
   - 计算现有赈灾粮可发天数
   - 估算还需借粮数量

2. **借粮执行**
   - 以县衙名义借粮
   - 知县出面，县衙担保
   - 记录借粮数量和归还期限

3. **分发赈济**
   - 设粥棚施粥
   - 每日两顿
   - 记录用粮数量

4. **后续处理**
   - 待官府拨粮后归还
   - 或以其他方式抵偿

## 决策点
- 若大户不愿借，可施压但需有度
- 借粮不足时需继续筹措

## 预期结果
- 灾民得到救济
- 防止民变发生
