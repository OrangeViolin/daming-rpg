# 明朝监斩行刑程序

## 概述
明朝官员执行死刑需严格遵守时辰和程序规定，监斩官需着官服，按《大明律》执行。

## 步骤

1. **准备阶段**
   - 监斩官换穿官服官帽
   - 在监斩台案前就座
   - 等待规定时辰（午时三刻）

2. **案卷审查**
   - 核对人犯口供
   - 查阅案卷文书
   - 确认无冤情后方可勾朱

3. **行刑准备**
   - 法号手、放铳手就位
   - 刽子手按刑种就位（火刑或绞刑）
   - 囚笼底板检查

4. **行刑执行**
   - 午时三刻准时行刑
   - 监斩官勾朱
   - 执行相应刑罚

## 决策点
- 若无案卷或口供，监斩官有权暂停行刑
- 若发现冤情，需上报上级衙门重新审理

## 预期结果
- 依法完成死刑执行
- 或暂停行刑等待上级裁决
