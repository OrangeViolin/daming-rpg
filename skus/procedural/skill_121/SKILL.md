# 皇帝勾决流程

## 概述
明朝皇帝对死刑犯名单进行勾决的法定程序，体现皇权对生杀大权的最终裁定。

## 步骤

1. **呈送名单**
   - 刑部将罪案呈奏皇上
   - 名单按顺序排列待决人犯姓名

2. **皇帝审阅**
   - 皇帝逐个审阅名单
   - 使用朱笔进行勾决

3. **勾决方式**
   - 勾决：在名字上划勾，表示批准执行
   - 划叉：表示发回重审或另有处置
   - 跳过：不勾决，表示暂缓或赦免

4. **送达执行**
   - 勾决名单送交内阁
   - 内阁转交刑部执行
   - 午时三刻行刑

## 决策点
- 皇帝是否勾决某一人犯？
- 是否有特殊旨意（如步行送达以延长时间）？

## 预期结果
- 死刑执行获得皇权批准
- 程序完整可追溯
