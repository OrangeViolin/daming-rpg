# 密旨召见外臣流程

## 概述
此流程用于皇帝秘密召见外地官员进京面圣，全程保密，不走驿站，不惊动他人。

## 步骤

### 一、发出密旨
1. 皇帝下密旨，规定进京时限
2. 密旨直接送达目标官员
3. 不经驿站，不走官方渠道

### 二、官员进京
1. 官员接旨后便服启程
2. 按规定时限（如五日内）抵达京城
3. 直接到指定朝房候见
4. 等候期间不进食，仅供应茶水

### 三、引领觐见
1. 太监到朝房轻声唤醒候见官员
2. 说明皇上正在等候
3. 引领官员前往觐见场所
4. 到达后通报："万岁爷，某某来了"

### 四、行礼入内
1. 官员在纱幔外跪下叩见
2. 皇上允准后方可入内
3. 若为特殊禁地，官员需表示不敢擅入
4. 太监传达皇上特许之意
5. 官员遵旨入内

### 五、面圣奏对
1. 按规制跪坐或站立
2. 回答皇上询问
3. 奉旨行事
4. 领旨后退出

## 决策点
- 若官员身体不支，可请旨赐座或赐食
- 若涉及机密，需确认无人知晓召见之事

## 预期结果
- 官员完成秘密觐见
- 旨意传达清楚
- 召见过程不外泄
