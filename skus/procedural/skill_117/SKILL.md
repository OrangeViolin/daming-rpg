# 奏疏撰写与封存流程

## 概述
此流程用于撰写重要奏疏并安全封存，确保内容保密。

## 步骤

### 一、草拟阶段
1. 确定奏疏主题和核心论点
2. 草拟奏疏内容
3. 反复推敲修改

### 二、誊写阶段
1. 准备正式纸张
2. 工整誊写全文
3. 标题写在顶头，如"治安疏"
4. 开头写明官职姓名，如"户部云南清吏司主事臣海瑞谨奏"

### 三、封存阶段
1. 将奏疏装入封套
2. 用饭团抹在封口处
3. 紧紧贴住封口，确保密封
4. 在封面上标注类别，如"贺表"

### 四、呈递准备
1. 等待催取人员上门
2. 亲手交付
3. 确保交接完成

## 注意事项
- 内容需深思熟虑，直指问题核心
- 封存需严密，防止提前泄露
- 标注类别可根据实际情况调整

## 预期结果
- 奏疏撰写完成
- 封存严密
- 按时呈递
