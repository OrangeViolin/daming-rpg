# 审讯取证方法论

## 概述
当犯人翻供、上级施压时，通过引入新证人、利用下属证词突破案件的审讯技巧。

## 步骤

### 一、让犯人翻供并记录
1. 不当场反驳犯人的翻供
2. 让犯人完整陈述翻供内容
3. 详细记录在案
4. 让犯人画押确认

### 二、引入关键证人
1. 找到犯人的直接下属或执行者
2. 收集书面证言（如士兵签名证词）
3. 证人出庭对质

### 三、区分公罪与私罪
1. **公罪**：奉命行事，可从轻或不追究
2. **私罪**：主动作恶，从严惩处
3. 利用此区分瓦解犯人防线

### 四、突破策略
1. 先问外围事实（如时间、地点）
2. 出示书面证言，打消抵赖念头
3. 明确告知：奉命行事是公罪，可从轻
4. 逐个击破，分别审讯

## 决策点
- 犯人翻供时：暂不反驳，先记录
- 证人犹豫时：告知公罪不究原则
- 证据不足时：收集书面签名证言

## 预期结果
- 犯人翻供内容被记录在案
- 下属证人提供关键突破
- 案件真相得以查明
