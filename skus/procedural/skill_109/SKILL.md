# 针灸急救流程

## 概述
针对突然昏厥、牙关紧闭、呼吸困难等急症，使用针灸进行紧急救治的方法。

## 步骤

1. **判断病情**
   - 观察患者面色（是否苍白如纸）
   - 检查牙关是否紧咬
   - 确认是否有呼吸和脉搏

2. **准备工具**
   - 取出银针包
   - 准备火源（烛台、火石火绒）
   - 准备艾叶卷
   - 准备沾有白药的棉球

3. **施针急救**
   - 点燃烛火
   - 将银针在烛火上烧灼消毒
   - 用药棉擦拭银针
   - 对准人中穴扎下银针

4. **艾灸辅助**
   - 点燃艾叶卷
   - 吹熄明火，保留暗火
   - 扒开患者衣襟
   - 对准胸前相关穴位进行熏灸

5. **观察反应**
   - 观察患者牙关是否松开
   - 等待患者吐出长气
   - 确认患者睁眼恢复意识

6. **后续处理**
   - 慢慢捻出银针
   - 将患者抬到床上平躺休息
   - 开窗通风保持空气流通

## 预期结果
患者恢复意识，呼吸平稳，可转入后续治疗。
