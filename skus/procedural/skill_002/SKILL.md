# 明朝年度财政会议流程

## 概述
明朝年度财政会议在宫中召开，由司礼监主持，内阁各部官员参与，审议上年度账目和来年开支。

## 会议步骤

1. **会前准备**
   - 内阁将去年各项开支按各部和两京一十三省实际用度报上来
   - 内阁拟好票拟
   - 司礼监准备批红

2. **会议召开**
   - 司礼监掌印太监主持会议
   - 内阁首辅定调子
   - 各部按顺序汇报

3. **审议流程**
   - 内阁报告各项开支
   - 户部核对账目并决定是否签字
   - 有疑问的票拟逐项讨论
   - 户部签字确认后，司礼监批红

4. **争议处理**
   - 户部可对超支账目提出疑问
   - 相关部门需解释说明
   - 皇帝或司礼监裁决

5. **决议形成**
   - 批红的票拟正式生效
   - 未签字的票拟需重新议定

## 决策点
- 户部是否在票拟上签字
- 司礼监是否批红
- 重大争议需皇帝裁决

## 预期结果
- 上年度账目结清
- 来年预算确定
- 各部职责明确
