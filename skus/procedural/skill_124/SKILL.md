# 诏狱管理流程

## 概述
明朝北镇抚司诏狱号称天下第一狱，管理严格，程序规范。

## 步骤

1. **收押程序**
   - 使用"虎狼套"（金步摇）刑具锁拿
   - 从头到脚全身披满锁链
   - 手脚铐在一起，只能一步一步挪动

2. **牢房配置**
   - 四面石墙，满地石面，顶上石板
   - 狱深地面一丈，常年不见日光
   - 设有石道、油灯照明

3. **日常管理**
   - 提供稻草铺地
   - 定时送饭
   - 狱卒看守

4. **审讯安排**
   - 可在牢房内审讯
   - 搬入桌椅供审案官使用
   - 门外专人记录

5. **处决准备**
   - 立秋日在诏狱大院设绞架
   - 梧桐树下设香案问神
   - 日晷判定时辰

## 决策点
- 是否给予特殊待遇（如桌椅床铺）？
- 是否允许探视？
- 处决前是否提供"断头饭"？

## 预期结果
- 囚犯安全看管
- 审讯顺利进行
- 处决程序完备
