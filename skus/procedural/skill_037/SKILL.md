# 案件审查程序

## 概述
明朝官员审理案件需依法审查案卷和口供，不得滥杀无辜。

## 步骤

1. **索阅案卷**
   - 要求查看完整案卷
   - 核对人犯抓获时间、地点
   - 检查禀报时间是否合理

2. **审查口供**
   - 要求提供人犯口供
   - 核实口供真实性
   - 发现矛盾需重新审理

3. **法律依据**
   - 依据《大明律》条款
   - 确认罪名是否成立
   - 检查程序是否合法

4. **上报程序**
   - 发现问题需向上级衙门呈报
   - 请求上级共同审理
   - 不得擅自处决

## 决策点
- 若无案卷口供，有权拒绝行刑
- 若发现冤情，需暂停并上报

## 预期结果
- 依法公正审理案件
- 防止冤假错案
