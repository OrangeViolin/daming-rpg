# 皇孙诞生赏赐流程

## 概述
皇室添丁是重大喜事，需按祖制进行赏赐和人员配置。

## 执行步骤

1. **报喜**
   - 接生确认婴儿性别
   - 太监持托盘盛放玉璋
   - 赴宫中向皇上报喜
   - 跪呈玉璋

2. **皇帝赐礼**
   - 皇帝赐予吉祥物品（如冬枣、栗子寓意"早立子"）
   - 众人颂圣

3. **照祖制赏赐**
   - 赏赐喜庆宝物
   - 调派太监二十名
   - 调派宫女二十名
   - 前往伺候

4. **通知相关人员**
   - 通知裕王师傅和侍读
   - 准许前往贺喜

## 预期结果
- 皇孙获得正式认可
- 赏赐和人员配置到位
- 相关人员获准庆贺
