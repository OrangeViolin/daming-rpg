# 明朝牢狱管理制度

## 概述
明朝各级牢狱有统一规制，需防范狱卒克扣牢粮、下毒灭口等行为。

## 步骤

1. **牢粮管理**
   - 官仓配拨牢粮
   - 分发前由差役先尝每碗饭
   - 防止掺沙石或下毒

2. **灯火管理**
   - 每日定量领取灯油
   - 牢房门口点大灯照明
   - 通道点小油灯

3. **安全防范**
   - 值房设于牢房通道出口
   - 差役挎刀守卫
   - 限制外人进入

4. **人犯看管**
   - 严加看管防止瘐毙
   - 防止走脱或被灭口

## 决策点
- 若发现牢饭有问题，立即停止分发
- 若上级派兵进入，监斩官有权拒绝

## 预期结果
- 人犯安全关押
- 防止冤案发生
