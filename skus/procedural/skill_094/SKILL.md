# 中药煎煮流程

## 概述
使用炭火炉煎煮中药汤剂的规范操作，确保药效最佳。

## 步骤

1. **准备器具**
   - 小炭炉一只
   - 药罐一个
   - 药碗一个
   - 竹筷两根
   - 铁钳

2. **开始煎药**
   - 将药材放入药罐
   - 加水适量
   - 点燃炭炉，用青色火苗加热
   - 盖上药罐盖

3. **控制火候**
   - 待白气直冲药盖发出叩动声
   - 揭开药罐盖，吹散白气
   - 用铁钳夹出部分红炭，降低火力
   - 将药盖搁在两根竹筷上
   - 用小火慢慢煎煮

4. **完成煎药**
   - 用布包住药罐把手
   - 将汤药滗入药碗
   - 轻轻吹凉后服用

## 注意事项
- 火候不可过猛，需慢火煎煮
- 煎药过程中需有人看守

## 预期结果
- 汤药浓度适宜
- 药效得到充分发挥
