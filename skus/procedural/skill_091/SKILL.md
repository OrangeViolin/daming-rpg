# 知县开堂审案流程

## 概述
明朝知县正式开堂审案的规范程序，从准备到判决的完整流程。

## 步骤

1. **升堂准备**
   - 听到堂鼓声后召集相关人员
   - 打开存放官服官印的木箱
   - 戴上官帽，穿上官服，系好腰带
   - 捧起官印走向大堂

2. **就位升堂**
   - 在大案前坐下
   - 三班衙役站班
   - 静坐不语，营造威严气氛

3. **传唤相关人员**
   - 命人传唤涉案官员或当事人
   - 等待人员到齐

4. **问案程序**
   - 以堂尊身份询问下属
   - 要求行下属之礼
   - 逐一质问案情
   - 要求出示相关公文证据

5. **查验公文**
   - 确认公文收件对象
   - 核实自己是否仍在职
   - 判断公文是否有效

6. **判决执行**
   - 引用大明律条文
   - 宣布罪名和刑罚
   - 抽签发令执行

## 决策点
- 是否接受公文有效性：需确认吏部是否有免职回文
- 是否当场处置：视情节轻重决定

## 预期结果
- 案件得到审理和判决
- 维护法纪和百姓利益
