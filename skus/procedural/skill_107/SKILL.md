# 传统荷叶包装技法

## 概述
使用荷叶和棕叶条包装食品的传统方法，既环保又美观，适合酱菜等食品的包装。

## 步骤

1. **准备材料**
   - 取晒干的大荷叶若干片
   - 准备素白瓷碗作为模具
   - 准备撕成条的棕叶备用

2. **铺底荷叶**
   - 抽出一片大荷叶
   - 将荷叶贴在瓷碗内壁作为衬底

3. **装填食品**
   - 揭开食品坛盖
   - 用漏眼勺舀出食品
   - 滗干汁液后倒入荷叶衬底的碗中
   - 重复直至装满

4. **包裹成型**
   - 将碗中的荷叶连同食品提出
   - 放到另一片更大的荷叶上
   - 飞快地将外层荷叶包好

5. **捆扎封口**
   - 从棕叶条中抽出三条
   - 在包装上一横一竖一斜绕成六合同心结
   - 扎紧后提起即可

## 预期结果
食品被荷叶完整包裹，棕叶捆扎牢固，便于携带且保持新鲜。
