# 政治危机应对策略

## 概述
面对上级或同僚的政治拉拢、施压时，如何保全自身、不落把柄的应对方法。

## 步骤

### 一、识别拉拢意图
1. 分析对方真实目的
2. 判断是否为陷阱
3. 评估接受或拒绝的后果

### 二、婉拒策略
1. **装聋作哑**：假装听不懂暗示
2. **引经据典**：用经典文献表达立场
3. **以退为进**：强调自己能力不足、年迈体衰
4. **坚守原则**：以体制、规矩为由拒绝

### 三、应对施压
1. 不直接对抗，但也不妥协
2. 用制度、规矩作为挡箭牌
3. 必要时示弱，但不失底线
4. 等待时机，不急于表态

## 决策点
- 对方以利益诱惑时：分析背后意图，谨慎应对
- 对方以威胁施压时：以退为进，不正面冲突
- 对方试探底线时：坚守原则，不轻易让步

## 预期结果
- 不被拉入对方阵营
- 不给对方留下把柄
- 保持自身清誉和立场
