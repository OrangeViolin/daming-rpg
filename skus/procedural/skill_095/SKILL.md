# 官员问责惩处流程

## 概述
依据大明律对违法官员进行问责、定罪和惩处的规范程序。

## 步骤

1. **列举罪状**
   - 明确具体违法行为
   - 引用相关法律条文
   - 每项罪名单独列出

2. **引用法律依据**
   - 查阅大明律相关条款
   - 明确罪名对应的刑罚
   - 如：行贿上司、忤逆不孝等

3. **宣布刑罚**
   - 明确刑罚种类（如杖刑、流放）
   - 明确刑罚程度（如杖四十、流三千里）

4. **给予申辩机会**
   - 听取被问责者陈述
   - 考虑是否有从轻情节

5. **执行或缓刑**
   - 根据态度决定是否立即执行
   - 可给予改正机会（如接回父亲奉养）

## 常见罪名及刑罚
- 挪用公款行贿上司
- 虐待父亲、忤逆不孝
- 杖四十、流三千里

## 预期结果
- 官员受到应有惩处
- 维护法纪威严
