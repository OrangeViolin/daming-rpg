# 三法司会审流程

## 概述
明朝重大案件由刑部、大理寺、都察院三法司正副堂官会同审理的法定程序。

## 步骤

1. **人员配置**
   - 刑部尚书、大理寺正卿、都察院左督御史为正堂官
   - 刑部侍郎、大理寺少卿、都察院左副都御史为副堂官
   - 共六人参与会审

2. **座位安排**
   - 正堂官坐在左侧大案前
   - 副堂官坐在右侧大案前

3. **记录要求**
   - 六人每人面前摆纸笔墨砚
   - 同时记录审讯内容
   - 审完后六份记录同时上呈比对

4. **审理程序**
   - 传唤人犯
   - 主审官发问
   - 人犯回话
   - 逐一记录

5. **定罪呈奏**
   - 会审后拟定罪名
   - 写成罪案呈奏皇上
   - 等待勾决

## 决策点
- 谁担任主审？
- 是否需要动刑？
- 罪名如何拟定？

## 预期结果
- 审理过程公正透明
- 六份记录一致可查
- 罪案呈奏符合程序
