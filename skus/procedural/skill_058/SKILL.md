## 概述
明朝司法审讯的规范流程，适用于省级以上衙门审理重大案件，确保审讯过程合法、口供有效。

## 步骤

### 一、审讯准备
1. 确认审讯权限（持有上谕或正式授权）
2. 安排审讯场所：明暗两间，明间审讯，暗间记录
3. 准备记录工具：口供纸、砚台、毛笔
4. 安排书办负责记录

### 二、提审犯人
1. 派狱卒将犯人带入明间
2. 根据犯人身份决定是否去刑具（未定罪官员可去锁链）
3. 根据情况安排座位（体现礼待或威慑）
4. 支退无关人员，关闭房门

### 三、正式审讯
1. 明确身份和审讯依据
2. 按圣旨或指控逐项提问
3. 要求犯人如实回答
4. 每项问答均要求书办记录在案
5. 对关键陈述要求大声重复确认

### 四、固定口供
1. 审讯完毕后，将口供交犯人阅看
2. 要求犯人在口供上画押
3. 如犯人拒绝画押，告知法律后果（《大明律》规定可立杖四十）
4. 必要时扶犯人到案边完成画押

## 决策点
- 犯人推诿时：追问具体细节，要求明确回答
- 犯人沉默时：提高声调，强调圣旨权威
- 犯人试图攀扯他人时：记录在案，继续追问核心问题

## 预期结果
- 获得犯人签字画押的有效口供
- 口供记录完整、清晰
- 审讯过程符合《大明律》规定
