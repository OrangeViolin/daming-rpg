# 明朝公文票拟批红流程

## 概述
明朝重要公文需经内阁票拟、户部审核、司礼监批红三道程序方可生效。

## 处理步骤

1. **内阁拟票**
   - 内阁根据事项拟定处理意见
   - 形成票拟文件
   - 送达相关部门

2. **部门审核**
   - 户部等相关部门核对账目
   - 核实无误后签字确认
   - 有疑问则暂不签字，提出异议

3. **争议协商**
   - 未签字的票拟需说明原因
   - 相关部门解释说明
   - 达成一致或上报裁决

4. **司礼监批红**
   - 户部签字后的票拟送司礼监
   - 司礼监用朱笔批红
   - 批注"照准"等字样

5. **归档执行**
   - 批红文件正式生效
   - 交相关部门执行
   - 存档备查

## 决策点
- 户部是否签字：依据账目是否核实清楚
- 司礼监是否批红：依据户部是否签字及皇帝旨意

## 预期结果
- 公文获得正式批准
- 各方责任明确
- 事项可付诸执行
