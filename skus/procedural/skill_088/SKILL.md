# 用人识人原则

## 概述
识别人才、合理用人的核心原则，确保关键位置由可靠之人担任。

## 核心原则

### 一、用人第一要义
> 用对了人才是干大事的第一要义

### 二、人才分类
1. **实心用事者**：忠诚可靠，委以重任
2. **顾全大局者**：识大体，可托付
3. **投机取巧者**：能力虽强，不可重用
4. **贪墨营私者**：坚决不用

### 三、用人策略
1. **留人手段**：
   - 让关键人才有存在价值
   - 制造依赖关系
   - 例：倭寇在，胡宗宪就在；胡宗宪在，谁也扳不倒我们
2. **用人不疑**：对选定的人才给予信任
3. **及时纠错**：发现用人不当立即调整

### 四、识人方法
1. 观其言行是否一致
2. 考察其在关键时刻的选择
3. 了解其过往履历和口碑
4. 观察其对待下属的态度

## 决策点
- 人才有争议时：看其核心品质和关键时刻表现
- 人才有过失时：区分无心之过与有意为之
- 需要换人时：果断调整，不可犹豫

## 预期结果
- 关键岗位由可靠之人担任
- 人才各得其所
- 团队稳定高效
