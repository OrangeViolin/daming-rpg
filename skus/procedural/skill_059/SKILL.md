## 概述
针对倭寇等敌军挟持平民的情况，实施陆海三路协同攻击的战术流程，核心目标是解救被挟百姓。

## 步骤

### 一、战前侦察
1. 使用千里镜观察敌寨和敌船情况
2. 确认敌军位置、火力配置
3. **关键判断**：确认敌军是否挟持平民
4. 确定平民所在位置（寨栅内、船板上等）

### 二、制定战术
1. 若敌军挟持平民：
   - 禁止直接炮击敌军核心区域
   - 制定包围合围计划
   - 准备近战突击方案
2. 部署三路兵力：
   - 陆路主力正面待命
   - 两路水师分别从东面、西南面迂回

### 三、实施攻击
1. **第一阶段**：水师就位形成合围
2. **第二阶段**：敌军开火后，我方炮火仅向敌军周边射击，避免伤及平民
3. **第三阶段**：利用天气条件（如大雾）派遣突击队近战登船
   - 选派熟悉水性的义兵或精锐
   - 配备快船、竹篙、抓钩长绳
   - 近距离登船肉搏
4. **第四阶段**：突击队得手后，三路同时发起总攻
   - 水师调整风帆逼近
   - 陆路骑兵、步兵全线出击

### 四、战后处置
1. 救出被挟持百姓
2. 生俘倭寇关押审讯
3. 缴获敌船物资
4. 妥善安置无家可归者

## 决策点
- 敌军炮火猛烈时：是否还击？若平民在敌阵中，坚持不直接还击
- 天气变化（起雾）：是否利用？是，可掩护突击队接近
- 突击队请求出战：评估其能力和风险后决定

## 预期结果
- 解救被挟持平民
- 歼灭或俘获敌军
- 我方伤亡最小化
