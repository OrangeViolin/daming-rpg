# 中医急救流程

## 概述
此流程用于处理因过度劳累、不食不睡、风寒侵入肌骨导致的昏厥重症。

## 诊断要点
- 患者极阳之体骤然受寒
- 几日几夜不食不睡
- 极阳尽而极阴生，风寒侵入肌骨

## 步骤

### 一、准备工作
1. 搬来所有棉被盖在患者身上
2. 搬来火盆，生一盆大火
3. 准备热粥备用

### 二、发汗疗法
1. 用多层棉被严密覆盖患者
2. 火盆加大火力，使室温升高
3. 等待患者额上见汗
4. 用手探入被中诊脉确认

### 三、艾灸治疗
1. 点燃艾灸
2. 拨开患者头顶头发，露出天灵穴
3. 对准天灵穴施灸
4. 观察患者反应

### 四、苏醒后调理
1. 患者张口吐出长气后慢慢睁眼
2. 替患者揩去汗水
3. 慢慢喂食热粥
4. 继续保暖休息

## 决策点
- 若发汗后仍未苏醒，需考虑其他病因
- 身病好医，心病难愈，需关注心理因素

## 预期结果
- 患者苏醒
- 风寒排出体外
- 恢复进食和行动能力
