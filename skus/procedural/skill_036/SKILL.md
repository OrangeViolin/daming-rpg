# 官员相貌选拔标准

## 概述
明朝取士除文章外，还要求相貌端正，所谓牧民者必有官相。

## 面形等级

### 第一等
- 国字脸
- 甲字脸
- 申字脸

### 次等
- 田字脸
- 由字脸

### 不合格
- 乃字脸 - 文章再锦绣，必然落榜

## 执行要点

1. 考试时附加相貌审查
2. 官帽一戴，便有官相
3. 无官相则无官威

## 预期结果
- 选拔出相貌端正的官员
- 确保官员有威仪治理百姓
