# 皇廷财政审计工作流程

## 概述
此流程用于集中核算大量财政账册，通过分组并行计算提高效率，确保核算结果准确且可追溯。

## 步骤

### 一、准备工作
1. 将待审计账册装箱，以加急方式运送至审计场所
2. 由内监签署收讫单子，确认接收
3. 准备两张长案，各置一把大型算盘
4. 从相关部门（如针工局、巾帼局、尚衣监）调派十二名精于算账的太监

### 二、人员分工
1. 将十二人分为两组，每组六人
2. 每组站于一张长案前，共用一把算盘
3. 明确职责：
   - 目光盯视账册
   - 左手拨弄算珠计算
   - 右手挥毫记录账目

### 三、核算执行
1. 太监从箱内依序取出账册，分送左右长案
2. 六人同时拨弄算盘，并行核算
3. 核算完毕后，记录结果于账单
4. 将账单吹干墨迹，双手呈交监审官

### 四、结果呈报
1. 监审官收集已核算账单
2. 在灯下仔细审阅
3. 依序摆放在御案上，按年份排列
4. 等待最终裁决

## 决策点
- 若发现账目异常，需标注并单独呈报
- 若核算未完成，继续并行计算直至全部完成

## 预期结果
- 所有账册核算完毕
- 账单按年份整齐排列于御案
- 贪墨数额清晰可查
