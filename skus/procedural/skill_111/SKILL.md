# 政治危机生存策略：思危、思退、思变

## 概述
在政治斗争中遭遇危机时，通过"思危、思退、思变"三步心法保全自身，等待时机。

## 核心原则

### 一、思危
- 时刻保持警惕，预判潜在危险
- 分析当前处境中的不利因素
- 识别可能的敌人和威胁来源
- 不因一时安稳而放松警觉

### 二、思退
- 危机来临时主动退让
- 接受降职、外放等不利安排
- 避免正面冲突，保存实力
- 退到相对安全的位置重新立足

### 三、思变
- 在退守中寻找转机
- 培养新的依靠和关系
- 等待时局变化
- 伺机重新崛起

## 具体应用步骤

1. **识别危机信号**
   - 上级态度变化
   - 同僚排挤
   - 职务被架空

2. **主动退让**
   - 不争辩、不反抗
   - 接受任何安排
   - 表现出顺从

3. **寻找新立足点**
   - 依附新的靠山
   - 建立新的关系网
   - 在新位置做出成绩

4. **耐心等待**
   - 忍受屈辱和困难
   - 不急于求成
   - 等待局势变化

5. **把握时机**
   - 当靠山得势时顺势而起
   - 当敌人失势时抓住机会

## 预期结果
在政治风暴中保全性命，积蓄力量，最终实现逆转或至少平安度过危机。
