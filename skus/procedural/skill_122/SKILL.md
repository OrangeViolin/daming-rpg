# 宫廷情报报告制度

## 概述
明朝帝王驭臣之术，通过提刑司、镇抚司侦知官员动向，建立日有日报、月有月报的情报体系。

## 步骤

1. **日常监控**
   - 提刑司、镇抚司负责侦控
   - 重点监控握有重权大臣的动向
   - 对有异常举动的中下层官员也派人布控

2. **记录行状**
   - 每日记录官员情状
   - 注明日期、时辰、具体行为
   - 格式："某年某月某日某时，某官员某行为"

3. **呈报制度**
   - 日有日报
   - 月有月报
   - 呈报至司礼监首席秉笔太监

4. **归档备查**
   - 所有呈报存档
   - 随时可调阅查证

## 决策点
- 哪些官员需要纳入监控范围？
- 发现异常行为是否立即上报？

## 预期结果
- 官员动向尽在掌握
- 情报系统运作有序
