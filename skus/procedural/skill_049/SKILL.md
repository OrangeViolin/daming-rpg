# 圣旨宣读流程

## 概述
圣旨宣读是庄重的政治仪式，需严格遵循规制，体现皇权尊严。

## 步骤

### 一、准备阶段
1. 太监从袖中取出圣旨
2. 递交给指定宣读者（通常为阁老）
3. 宣读者站立准备

### 二、接旨就位
1. 宣读者宣布："有旨，某某跪听旨意"
2. 接旨者走到堂中正中位置
3. 双膝跪下，双手扶地
4. 低头静候

### 三、宣读圣旨
1. 宣读者展开圣旨
2. 以庄重缓慢的声调宣读
3. 开头："奉天承运，皇帝诏曰"
4. 正文：逐字逐句清晰朗读
5. 结尾："钦此"

### 四、领旨谢恩
1. 宣读者读完最后一个字
2. 接旨者磕头
3. 口称："臣领旨谢恩"
4. 等候起身指令

### 五、后续处置
1. 接旨者起身
2. 若有后续安排，听从指示
3. 若被免职，立即退出

## 决策点
- 若圣旨内容出人意料，宣读者需保持镇定
- 若多人同时接旨，需逐一确认谢恩

## 预期结果
- 圣旨宣读完毕
- 接旨者明确旨意内容
- 仪式庄重规范
