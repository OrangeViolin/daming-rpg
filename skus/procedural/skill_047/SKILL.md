# 内阁会议规制

## 概述
内阁会议有严格规制，座位安排、发言顺序均体现政治格局，任何变化都意味着权力调整。

## 步骤

### 一、会场布置
1. 首辅居中而坐
2. 其他阁员按地位分坐左右
3. 若有太监代表，坐于首辅左侧
4. 若有变化，需调整座位摆放

### 二、入场就座
1. 与会者按规制入座
2. 观察座位变化判断会议性质
3. 保持安静，等待首辅或宣旨者开口

### 三、议事程序
1. 首辅或主持者先发言
2. 阁员按顺序发表意见
3. 若有争议，不可互相打断
4. 不可失态咆哮

### 四、宣旨环节
1. 若有圣旨，由指定阁老宣读
2. 接旨者走到堂中跪下
3. 宣读者站立宣读
4. 接旨者听完后磕头谢恩

### 五、议决事项
1. 就人事任免等事项进行讨论
2. 各方可举荐人选
3. 最终由首辅或宣旨者定夺
4. 拟票形成决议

## 决策点
- 若座位有变，需判断政治风向
- 若被要求退出，需立即领旨谢恩离开

## 预期结果
- 会议按规制完成
- 人事调整或政策决议形成
- 各方明确自身处境
