# 公文有效性核验流程

## 概述
收到上级公文时，核验其是否有效、是否应当执行的判断程序。

## 步骤

1. **确认收件对象**
   - 查看公文抬头写明给谁
   - 是发给个人还是发给衙门

2. **核实官员任职状态**
   - 检查是否有吏部正式免职回文
   - 确认现任官是否已正式离任
   - 辞呈递交不等于免职生效

3. **判断公文效力**
   - 若官员仍在任，公文应交给该官员
   - 若官员已正式免职，公文可交给代理者

4. **决定执行与否**
   - 公文内容是否合法合理
   - 是否与现行政策冲突
   - 是否损害百姓利益

## 决策点
- 官员是否已正式免职：以吏部回文为准
- 公文内容是否正当：可拒绝执行不当命令

## 预期结果
- 正确判断公文效力
- 避免越权或失职
