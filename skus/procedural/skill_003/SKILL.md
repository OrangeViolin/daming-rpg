# 宫廷祥瑞报喜流程

## 概述
宫廷中发现祥瑞（如瑞雪）后，需按特定程序向皇帝报喜，并逐级传达。

## 报喜步骤

1. **发现祥瑞**
   - 当值人员发现祥瑞现象
   - 确认祥瑞性质（如瑞雪、灵芝等）

2. **初步控制**
   - 禁止其他人随意声张
   - 指定专人负责报喜

3. **逐级上报**
   - 先向皇上报喜
   - 再向司礼监报告
   - 司礼监掌印太监知晓

4. **正式宣布**
   - 司礼监正式向全场宣布祥瑞
   - 允许众人欢呼庆贺
   - 组织相关庆祝活动

## 注意事项
- 报喜顺序不可颠倒
- 未经许可不得擅自声张
- 违者可能受罚

## 预期结果
- 皇帝获知祥瑞并龙颜大悦
- 宫廷上下共同庆祝
- 祥瑞被记录在案
