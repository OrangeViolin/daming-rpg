# 借天气制定军事策略流程

## 概述
利用天气变化规律，结合河图洛书的哲学思想，制定有利的军事作战计划。

## 步骤

1. **观察天气变化**
   - 关注当前天气状况（如暴雨）
   - 根据经验判断后续天气（如大雨后必有大雾）

2. **分析战术优势**
   - 大雾天气有利于奇兵突袭
   - 敌人视野受限，我方可隐蔽行动

3. **确定作战时机**
   - 等待雨停雾起
   - 选择最佳出击时间

4. **部署兵力**
   - 通令各路援军
   - 全线出击准备

5. **发起总攻**
   - 雨停雾起时全线出击
   - 一举聚歼敌军

## 决策依据
- 历年作战经验总结的天气规律
- 河图洛书"应天时而动"的原理

## 预期结果
- 利用天气优势取得战术胜利
- 减少己方伤亡，提高作战效率
