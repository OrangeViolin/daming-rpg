# 锦衣卫选拔标准

## 概述
锦衣卫选人有严格身体条件要求，需具备特殊体能和格斗能力。

## 身体条件

1. **虎臂** - 肩膀特别宽
2. **蜂腰** - 腰部被带子束紧显得特别细
3. **螳螂腿** - 腿部青筋暴露，硬如铁柱

## 技能要求

1. **擅走** - 一人每天能走一百六十里以上
2. **擅跳** - 两丈高的墙，跃起双手一攀，翻身便能过去
3. **擅斗** - 有拳脚兵器功夫，更要有狠劲
4. **马功**（最高要求）- 七天七晚不坐不躺，两条腿轮流踩在地上睡觉，七天头上双脚着地还能空手杀死一头狼

## 预期结果
- 选拔出符合标准的精锐人员
- 组建执行特殊任务的队伍
